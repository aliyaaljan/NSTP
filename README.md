# NSTP

Welcome to the NSTP project! This is a Next.js application. Follow the instructions below to set up your environment and run the project locally.

---

## Prerequisites

Before you begin, make sure you have the following installed:

- Node.js (version 18.17 or later) — https://nodejs.org/
- A package manager: npm (comes with Node.js), yarn, or pnpm

---

## Getting Started

### Step 1 — Clone the repository

```
git clone https://github.com/aliyaaljan/NSTP.git
cd NSTP
```

### Step 2 — Install dependencies

```
npm install
```

You can also use `yarn install` or `pnpm install` if you prefer.

### Step 3 — Run the development server

```
npm run dev
```

Then open your browser and go to: http://localhost:3000

The page will auto-update as you edit and save files inside the `app/` folder.

---

## Available Commands

| Command | What it does |
|---|---|
| `npm run dev` | Starts the local development server |
| `npm run build` | Builds an optimized production version |
| `npm start` | Runs the app in production mode (run build first) |
| `npm run lint` | Checks your code for errors and style issues |
